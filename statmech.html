Parameters Description
vib_energies ((N,) numpy.ndarray) Vibrational energies in eV

(str) Geometry of molecule. Supported options:

  • monatomic
  • linear
  • nonlinear
potentialenergy (float) Potential energy in eV

(int) Symmetry number

Point group symmetry number

See DOI for more details: 10.1007/s00214-007-0328-0
spin

(float) Holds the total electronic spin

  • 0 for molecules in which all electrons are paired
  • 0.5 for a free radical with a single unpaired electron
  • 1.0 for a triplet with two unpaired electrons, such as O_2

class PyMuTT.models.statmech.hinderedthermo.HinderedThermo(vib_energies, trans_barrier_energy, rot_barrier_energy, sitedensity, rotationalminima, potentialenergy=0.0, mass=None, inertia=None, atoms=None, symmetrynumber=1)¶

Hindered translator/hindered rotor model. Used for surface species where there are 2 translational modes, 1 rotational mode perpendicular to the surface, and 3N-3 vibrational modes.

See further documentation from ase.thermochemistry.HinderedThermo

ase.thermochemistry.HinderedThermo

rotationalminima (int) Number of equivalent minima for an adsorbate full rotation e.g. 6 for an adsorbate on an fcc(111) top site
